# LoVisual Launcher

Rust + egui desktop launcher for LoVisual client.

## Build

```bash
cargo build --release
```

## Features

- Login/Register via backend API
- Profile dashboard with playtime
- Settings panel for custom launch command
- Launches official Minecraft launcher or Fabric directly
- Saves session to cross-platform config dir

## Usage

1. Run `lovisual-launcher`
2. Login or Register
3. Click PLAY to launch Minecraft with the mod
